## Why Prompt Craft?

Most people struggle to get consistent, high-quality responses from AI tools. The problem isn't the AI—it's unclear prompts. Prompt Craft solves this by teaching you the CLEAR Framework and helping you apply it to every prompt you write.

**The goal:** Get 80% of what you want on the first try.

## The CLEAR Framework

Every great prompt has five essential elements:

- **C**ontext - Set the stage with role, situation, and background
- **L**ength - Specify the desired output format and word count
- **E**xamples - Show the style and structure you want
- **A**ction - Use strong, specific verbs that describe the task
- **R**equirements - Provide a detailed checklist of what to include
